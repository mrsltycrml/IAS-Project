# **App Name**: School Showcase

## Core Features:

- Home Page Display: Display photo in the upper left corner, along with a short description or header for the site.
- Notes Overview: Display all notes on the home page for a quick overview.
- Selected Notes Viewer: Allow users to view selected notes in detail on a separate page.
- About Me Page: Dedicated page showcasing 'about me' information.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trustworthiness and competence.
- Background color: Light gray (#F0F4F8), almost white, for a clean and professional look.
- Accent color: Deep blue (#3F51B5), to direct the user's eye to important elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable style.
- Clean, single-column layout to emphasize simplicity and focus on content.
- Simple and professional icon set (e.g. Font Awesome) for easy navigation.
- Subtle fade-in animations on page load to create a smooth user experience.
